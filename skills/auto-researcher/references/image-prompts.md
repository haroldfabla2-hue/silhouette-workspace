## Infographic Prompts

### Modern Infographic
```
Professional infographic about: {topic}

Visual elements:
- Header with title
- 3-5 key statistics with icons
- Brief explanatory text
- Trend indicator (up/down/stable)

Style: Modern, clean, information-dense
Color scheme: Professional blue (#1a365d) and gray
Layout: Vertical flow, single column
Font: Sans-serif, readable
```

### Data Visualization
```
Data-rich visualization about: {topic}

Include:
- Main chart or graph
- 2-3 supporting statistics
- One-sentence insight
- Source citation

Style: Academic/professional
Colors: Navy and white with one accent
Format: Square or 4:3 aspect ratio
```

### Process Flow
```
Flowchart showing: {process_steps}

Style: Clean diagram
Elements: Boxes with arrows
Labels: Clear step descriptions
Color scheme: Green for completed, blue for current
```

## Tips for Better Results

1. **Be specific**: Include exact data points when possible
2. **Set expectations**: "Minimalist" vs "detailed"
3. **Specify format**: "Vertical bar chart" not just "chart"
4. **Color matters**: Use professional palettes
5. **Keep it simple**: 3-5 elements maximum
